# ロジコムトランスポート株式会社 コーポレートサイト

埼玉県川口市の物流会社「ロジコムトランスポート株式会社」のコーポレートサイト（静的HTML）。

GitHub Pages で公開しています。

## ページ構成

| ファイル | 内容 |
|---------|------|
| `index.html` | トップページ |
| `company.html` | 会社案内（代表挨拶・会社概要・沿革） |
| `service.html` | 事業案内（ルート配送・チャーター・倉庫保管・流通加工） |
| `vehicles.html` | 車両・設備 |
| `recruit.html` | 採用情報 |
| `contact.html` | お問い合わせフォーム |
| `access.html` | アクセス・地図 |

## 技術メモ

- フレームワーク不要のプレーンな静的HTML + バニラJS。ビルド不要。
- Claude Design のプロトタイプ（`.dc.html` / React ランタイム）から、CDN依存のない静的サイトへ変換したもの。
- スクロールで要素が現れるアニメーションは `IntersectionObserver`、ホバー効果は CSS で実装。
- フォント: Google Fonts（Zen Kaku Gothic New / Bebas Neue）。

## お問い合わせフォームについて（要対応）

`contact.html` のフォームは**現状クライアント側の入力チェックのみ**で、送信内容はどこにも送られません。
実際に問い合わせを受信するには、以下のいずれかの対応が必要です。

- [Formspree](https://formspree.io/) など外部フォームサービスに接続する
- サーバー側の送信処理（メール送信API等）を用意する

それまでは、フォーム下部に記載の電話（048-229-5801）／FAX（048-229-5811）が実際の連絡手段です。
